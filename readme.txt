Здесь находится код, с помощью которого я выбрал вариант архиетктуры для нейросети, восстанавливающей смазанные изображения, и обучал этот вариант.
В файле train2 происходит выбор варианта архитектуры нейронки, а в файле untitled4 его обучение. В файле motion_modeling происходит моделирование смаза спутниковых изображений. В файле inverse_and_wiener - их восстановление с помощью инверсного фильтра и фильтра Винера
